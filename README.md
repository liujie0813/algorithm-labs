## leetcode

重点：
- program：纯编程类
- array：数组
- linkedlist：链表
- stask_queue：栈、队列
- recursion：递归
- sort：排序
- binary_search：二分查找
- hashtable：哈希表
- string：字符串
- tree：树
- bst：二分搜索树
- bfs&dfs：宽度&深度
- dp：动态规划
- 回溯

题型：
- 单调栈
- 双指针、滑动窗口
- 前缀后缀统计
- 位运算
- 海量数据处理

其他：
- 字符串匹配
- 图相关算法
- 跳表
- 并查集
- Trie
- 贪心

## 二分查找

二分法要考虑的点：
- 是否有序数组；
- 是否有重复元素；
- 对区间的定义是什么
  - 左闭右闭 [left, right]
  - 左闭右开 [left, right)
